## sistema-editor-texto

Sistema para gravar e salvar textos ou fazer anotações, com telas visuais, ultilizando a classe Swing este sistema cria um arquivo txt, de acordo com o local que voce referencia no codigo.  

Ele é semelhando ao bloco de notas, voce digita sua informação, e quando pede pra salvado ele gera um txt no local,
referenciado por voce.
